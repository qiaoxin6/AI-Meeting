# xunzhi-debug-playbook

## 使用顺序

- 先按症状选剧本，不要先猜根因。
- 面试卡住看 `references/interview-stuck.md`。
- 运行时超时或重复看 `references/ai-runtime-failure.md`。
- 转写、WebSocket、TTS 看 `references/media-debug.md`。
- 恢复失败、只读恢复、结果丢失看 `references/restore-failure.md`。
